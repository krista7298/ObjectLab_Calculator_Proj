# Project Name: Calculator Application

## Project Description

This project is a console-based calculator application developed in C# using object-oriented programming principles. The application allows users to perform addition, subtraction, multiplication, and division through a menu-driven interface. It also includes memory features for storing calculation results and managing a collection of integers. The project demonstrates object-oriented design, exception handling, Lists, methods, properties, and user interaction.

## Project Tasks

- Task 1: Set up the development environment
  - Create the project in Visual Studio Code
  - Configure the GitHub repository
  - Verify the project builds successfully

- Task 2: Design the application
  - Plan the menu-driven interface
  - Identify the required calculator features
  - Separate the application into Calculator, Memory, and Program classes

- Task 3: Develop the Calculator class
  - Create methods for addition
  - Create methods for subtraction
  - Create methods for multiplication
  - Create methods for division
  - Validate division by zero

- Task 4: Develop the Memory class
  - Store a single calculation result
  - Store up to ten integer values in a List
  - Add, remove, retrieve, and clear stored values
  - Calculate the count, sum, average, and difference of stored integers

- Task 5: Develop the Program class
  - Create the main menu
  - Accept and validate user input
  - Call methods from the Calculator and Memory classes
  - Display calculation results and memory statistics

- Task 6: Implement exception handling
  - Handle invalid numeric input
  - Prevent division by zero
  - Display user-friendly error messages

- Task 7: Test the application
  - Test all calculator operations
  - Test all memory functions
  - Test exception handling
  - Debug and correct any errors

- Task 8: Document the project
  - Create a README file
  - Describe the project structure
  - Explain the purpose of each class
